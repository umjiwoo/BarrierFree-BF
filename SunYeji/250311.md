## Frontend 기능

### 입력

1. **제스처 입력**
    - 문자 입력 (드로잉)
        - 숫자 입력 → 비밀번호, 계좌번호 (ex. `2` `1` `3` → 213)
        - 정해진 제스처 → 메뉴 이동 (ex. `♡` → 송금 / `ㅅ` → 송금)
    - 스와이프, 드래그, 멀티터치
        - 위로 스크롤
        - 다음 페이지
        - 
2. 터치 입력
3. 모션 입력 (흔들기, 기울기)
4. + 음성 입력

### 출력

1. 햅틱 반응
2. 진동
3. 음성
